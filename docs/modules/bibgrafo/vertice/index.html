Vertice

Vertice(
   rotulo: str, attr = None

Esta classe representa um vértice de um grafo.

Attributes

Methods:

.rotulo

source

.rotulo()

str: O rótulo (ou nome) do vértice.

.attr

source

.attr()

dict: Um dicionário de atributos do vértice. De acordo com a modelagem ques e deseja fazer, vértices podem ter atributos não previstos nessa implementação. Dessa forma, foi incluído um dicionário de atributos para que o usuário possa incluir outros atributos que desejar.
